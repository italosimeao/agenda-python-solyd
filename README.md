# 📔 Agenda de Contatos em Python

Projeto final desenvolvido durante o módulo de **Introdução ao Python e Algoritmos** na plataforma Solyd One.

## 🚀 Funcionalidades
- Interface interativa via terminal com sistema de menu estruturado.
- Operações completas de CRUD: Criar, Buscar, Editar e Excluir contatos da memória.
- Sistema de persistência de dados automatizado com salvamento e carregamento de arquivos (`database.csv`).
- Tratamento rigoroso de exceções (`try/except/KeyError`) para evitar quebra do programa durante a execução.

## 🛠️ Tecnologias Utilizadas
- Python 3
- Manipulação nativa de arquivos (Built-in I/O)
- Estruturas de Dados Avançadas (Dicionários Aninhados)
